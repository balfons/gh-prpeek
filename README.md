Show status of relevant pull requests live


To install dependencies:

```bash
bun install
```

To run:

```bash
bun start --repo <owner>/<repository>
```

To compile:
```bash
bun compile
```

Run compiled executable:
```bash
./prpeek --repo <owner>/<repository> --sound --interval 15 --labels bug,feature
```
